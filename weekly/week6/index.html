4) Float Layout

The image is float: left, so this text wraps around it. Floats are old-school layout, but they’re still useful for simple text wraps. The .clearfix utility prevents this floated image from collapsing the parent’s height.

Tip: prefer modern layout (Flexbox/Grid) for most page structure. Floats are for text wrap moments.
